# 🎬 FUTURE_FS_03 - CineVerse Multiplex

A premium and modern cinema booking website built using React.js and TypeScript. CineVerse Multiplex provides movie enthusiasts with a seamless experience to explore movies, view show timings, book seats, read reviews, and discover upcoming releases through an elegant and luxurious user interface.

🌐 Live Server: https://cineversemultiplex.netlify.app/
🔗 GitHub: https://github.com/RigvedaChowdary  

## 📌 About the Project

CineVerse Multiplex is a modern digital platform designed for cinema businesses to strengthen their online presence and provide customers with a smooth movie discovery and booking experience.

The website allows users to:

* Browse currently showing movies
* Explore upcoming releases
* View detailed movie information
* Book seats through an interactive interface
* Read customer reviews
* Access cinema locations and contact information
* Experience a premium, mobile-friendly user interface
